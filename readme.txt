== Description ==

ปลั๊กอินตัวนี้ทำมาเพื่อให้ มีicon Social ต่างๆ อยุ่ในตำเเหน่งที่สวยงาม เเละ คนเข้าเวปจะได้ กดเพื่อติดต่อเราง่ายๆนะคับ


== Installation ==

วิธีการติดตั้ง ตาม ข้าง ล่าง

1. Upload plugin ทั้งfolder  ไว้ที่ `/wp-content/plugins/' หรือ กด install เเล้ว activate จาก Wordpress Plugin
2. เข้าไป ที่ menu  " Social Side"
3. ตั้งค่า ตามตัวอย่างด้านล่าง
